# Harry A. Zarcadoolas - Personal Website

Welcome to the repo for my personal resume website. This project is designed to provide an online presence for my professional profile and help showcase my skills as a computer engineer and full stack developer. It includes sections such as About Me, Skills, Experience, Projects, and Contact Information.

## Technologies Used

- HTML
- CSS
- JavaScript
- **React + Vite** (_front-end_)
- **Axios** (_front-end & back-end_)
- **Node.js + Express** (_back-end_)
